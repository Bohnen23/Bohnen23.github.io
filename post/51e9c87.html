Mac 系统下 Hexo+Github Blog（1） 搭建博客 小白遇坑记录

本文最后更新于：2023年8月6日 凌晨

用 Mac 前，曾用 Windows 搭建过，不得不说，Mac 明显比 Windows 简单，步骤如下：

创建本地环境

1、安装 Node.js

2、安装 Git

这两个步骤不赘述，检验是否安装成功，命令如下：

node -v

git -v

分别显示版本信息，证明已经安装成功。

3、安装 Hexo

打开 Mac 的 Terminal 页面（找不到的可以电脑搜索，或者 google 方法）

Step 1，在 Terminal 页面中输入如下命令：

sudo npm install hexo-cli -g

这里有一个坑

大多数引导用的命令是：

npm install hexo-cli -g

而我用这个一定报错，加上 sudo 就顺利完成。假如加 sudo 报错的话，就去掉试试。

Step 2，创建并初始化博客项目 —— Blog （这个名字可以换其他，比如 Myblog，那么下面 cd 后名字和这个要一致）

hexo init Blog

Step 3，切换到博客项目 —— Blog 中（如果上面取了 Myblog，这里就是 cd Myblog）

cd Blog

Step 4，安装

sudo npm install

跟上面一样，假如报错，就去掉 sudo

4、本地预览

生成静态网站，并在本地预览，命令如下：

hexo g --debug

hexo s

这时会生成一个网站 http://localhost:4000
